- `residential.html`
- `commercial.html`
- `fire-alarm.html`
- `security-systems.html`

- `access-control.html`
- `monitoring.html`
- `contact.html`

## Deploy

1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to Settings -> Pages.
4. Select Deploy from branch.
5. Select `main` and `/root`.
6. Save.

## Custom domain

When ready, rename `CNAME.example` to `CNAME` and ensure it contains:

`firebrandsystems.com`

Then configure DNS at the registrar and enable HTTPS in GitHub Pages.
